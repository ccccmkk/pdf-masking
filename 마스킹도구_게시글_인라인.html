PDF 개인정보 마스킹 도구 배포

안녕하세요, 서울서부지청 김창민 주무관입니다.

정보공개 업무를 하시거나 회의자료 준비 하실 때 PDF 파일의 개인정보 마스킹 처리를 편하게 하실 수 있도록 만들었습니다.

아래 기능을 읽어보시고 필요하신 분들은 편하게 사용해 주시면 됩니다.

* 본 프로그램은 Claude를 통해 바이브코딩으로 만들었습니다.

[다운로드링크(G드라이브)]
HTML 파일 하나만 받아서 더블클릭 → 브라우저에서 바로 사용

✅ 주요 기능

🎨 마스킹
드래그로 영역 선택 → 즉시 적용
검정·흰색·빨강·파랑·모자이크·커스텀 색상(스포이드 포함)
우클릭 삭제 / Ctrl+Z 되돌리기 		💾 저장
저장 즉시 이미지로 구워진 버전으로 교체 → 추가 마스킹 가능
페이지 선택 저장 / 다른 이름으로 저장
🔄 페이지 회전
현재 페이지 / 전체 좌우 90° 회전
저장·인쇄 시 회전 방향 그대로 반영 		🔍 확대/축소
화면 배율 50%~400% 조절
회전 상태 유지하며 확대 가능
📌 페이지 목록
우측 썸네일 패널 — 마스킹·회전 즉시 반영
핸들 드래그로 패널 크기 조절 		🖨️ 인쇄 / 페이지 이동
마스킹 상태 그대로 인쇄 다이얼로그
페이지 번호 입력으로 즉시 이동

📖 사용 방법

1 파일 열기 — 버튼 클릭 또는 PDF 파일을 창에 드래그&드롭

2 마스킹 — 가리고 싶은 영역을 마우스로 드래그 / 우클릭으로 삭제 / Ctrl+Z로 되돌리기

3 저장 — 💾 저장 버튼 클릭 → masked_원래파일명.pdf 로 자동 저장

💡 보안 모드(기본)로 저장하면 텍스트 레이어가 완전히 제거됩니다.
마스킹 영역뿐 아니라 문서 전체가 이미지로 변환되어 복사·파싱이 불가능합니다.
